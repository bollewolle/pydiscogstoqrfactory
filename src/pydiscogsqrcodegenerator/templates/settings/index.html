Printer Offset Correction Experimental

Note: This feature is experimental and may not produce accurate results with all printers.

If your printed stickers don't align with the sticker sheet, your printer may have an origin offset. Print a test page at 100% scale and measure the difference between the expected and actual margins. Use positive values to shift content up (top) or left (left). For example, if the top margin prints as 28mm but should be 26mm, set the top offset to 2.

Print Test Page

Download a test page PDF for the active sticker layout. It shows all layout dimensions (margins, sticker size, spacing) so you can print it at 100% scale and measure with a ruler to determine the correct printer offset values. Make sure to save your settings first if you changed offset values.

Tip: Finding dimensions for your sticker sheets

Microsoft Word has a built-in label database with dimensions for many brands and types (Avery, ULINE, etc.). To look up your sticker sheet dimensions:

  1. Open Microsoft Word and create a new blank document.
  2. Go to Mailings > Labels (on older Mac versions: Tools > Labels).
  3. Click Options, then select your label vendor and product number.
  4. Click Details to see the exact dimensions (margins, sticker size, spacing) that you can enter in the layout fields below.

You can also use Word to print a sheet with label outlines for visual verification: after selecting your label vendor and product number, click New Document to generate a page with a table matching the label grid. Then select the table, add visible borders (right-click > Table Properties > Borders and Shading), and print it. You can then place this printout under your actual sticker sheet to check alignment, or print a test QR PDF on top of it to verify that QR codes and text fit within the sticker boundaries.
